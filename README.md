# db

**db** is a generic database wrapper around [gorm](https://github.com/go-gorm/gorm).

## Installation

```sh
go get crdx.org/db
```

## Usage

Check out the documentation on [pkg.go.dev](https://pkg.go.dev/crdx.org/db).
